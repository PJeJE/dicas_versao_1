    ASPJE - Página de Dicas

    Principais arquivos

    docs/ - Pasta principal
    index.md - Página inicial

    recurso.md - Informações sobre recursos

    dicas.md - Página de dicas

    img/ - Coloque suas imagens nesta pasta

    Editando a página

    1. Acesse o endereço: https://github.com/PJeJE/dicas
    2. Faça login com o usuário da aspje.
    3. Clique na pasta docs.
    4. Clique no arquivo dicas.md.
    5. Clique no ícone do lápis (fica no cantor superior direito).
    6. Edite o arquivo como desejar pela aba Edit file.
    7. Pela aba Preview changes, você pode visualizar como ficou antes de salvar as alterações definitivamente.
    8. Para salvar, desça a tela até ao final. Existe uma seção chamada Commit changes.
    9. No primeiro campo, descreva de forma sucinta o que você alterou/incluir no arquivo.
    10. Deixe marcada a opção Commit directly to the master branch.
    11. Clique no botão Commit changes.
    12. Pronto! Dentro de 3 minutos a nova versão estará no ar!

    Incluindo uma imagem

    ![Texto alternativo](img/imagem.jpg)

    Exemplo:

    Obs. 1: A imagem deve estar na pasta img.

    • Para enviar uma nova imagem do seu computador:
      1. Clique na pasta docs.
      2. Clique na pasta img.
      3. Clique no botão Add File.
        • Selecione Upload files.
      4. Arraste a imagem do seu computador para a área indicada.
      5. Para salvar, desça a tela até o final, se necessário. Existe uma seção chamada Commit changes.
      6. No primeiro campo, descreva de forma sucinta o que você alterou/incluir no arquivo.
      7. As demais opções não precisam ser alteradas.
      8. Clique no botão Commit changes.
      9. Pronto!

    Obs. 2: Atenção ao tamanho da imagem! Redimensione se necessário antes de incluí-la.

    Obs. 3: O "texto alternativo" só aparece caso o navegador não consiga carregar a imagem ou é utilizado por programas de acessibilidade (Ex. Aplicativo que lê a página para cegos).
